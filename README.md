## Описание справочников

### Справочник 1: Бренды косметики
Хранит информацию о косметических брендах.

Поля:
- id
- brand_name
- country
- foundation_year
- website
- description

### Справочник 2: Косметические товары
Хранит информацию о косметических товарах.

Поля:
- id
- product_name
- brand_id
- category
- price
- manufacture_date
- quantity
- description

## Зависимость справочников
Справочник «Косметические товары» зависит от справочника «Бренды косметики».

Каждый товар обязательно связан с определенным брендом через поле brand_id.

## Используемые типы данных
В проекте использованы все обязательные типы данных:

- Текст:
  - название бренда
  - страна
  - сайт
  - категория товара

- Многострочный текст:
  - описание бренда
  - описание товара

- Целые числа:
  - id
  - год основания
  - количество товара

- Числа с фиксированной запятой:
  - цена товара

- Дата:
  - дата производства товара
